# Mahzaib Khalid Academic Portfolio

This is a static GitHub Pages portfolio for publishing profile details, education, publications, research projects, skills, and contact links.

## Customize

- Edit `index.html` to replace placeholder emails, profile links, university names, publication titles, and project details.
- Replace `assets/research-banner.png` with your own academic photo or banner if you prefer.
- Update colors and spacing in `styles.css`.

## Publish On GitHub Pages

1. Create a GitHub repository named `your-username.github.io` or any normal repository.
2. Upload these files to the repository.
3. In GitHub, open **Settings > Pages**.
4. Choose the branch, usually `main`, and the root folder.
5. Save, then open the published Pages URL after GitHub finishes deploying.
